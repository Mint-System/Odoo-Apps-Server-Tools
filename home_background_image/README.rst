=====================
Home Background Image
=====================

Set a background image for the Odoo company.

For a detailed documentation have a look at https://www.odoo-wiki.org/.

.. image:: https://raw.githubusercontent.com/Mint-System/Wiki/master/assets/icon-box.png
  :height: 100
  :width: 100
  :alt: Icon

Configuration
~~~~~~~~~~~~~

* Open the company form and click *Edit*
* Set a home backround image and click *Save*
